# Notebook

A simple note-taking application built with Python (PyQt5) and QML.

## Features

- Create, edit, and delete notes
- Dynamic line numbering for notes
- Scrollable text area for long content
- Search across notes with line-specific results
- Select a search result to open the corresponding note
- Prevents deleting the last remaining note
- Clean UI with rounded corners and responsive layout

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KostiantinKorchuhanov/Note_app.git
   cd Note_app
   ```
2. Install dependencies:
   ```bash
   pip install PyQt5
   ```
3. Run the application:
   ```bash
   python main.py
   ```
